Prettier配置选项说明

发表于2021-05-22|更新于2021-12-27|work
|字数总计:692|阅读时长:2分钟|阅读量:

Info: .prettierrc配置

配置说明

配置选项 技能 默认值 备注
printWidth 单行代码的最大长度 80 最大长度一般会设置成100-120，不建议太长，影响可读性
proseWrap 指定文件折行方式 preserve always:当超出printWidth时折行
never:永不折行
preserve:按照文件原样折行
tabWidth 指定缩进的长度 2
useTabs 指定使用tab还是space来进行缩进 false 为true时使用tab来进行缩进
semi 在末尾是否加分号 true false只会在可能导致ASI错误的行首添加分号
trailingComma 是否在可能的多行中输入尾逗号 es5 none:无尾逗号
es5:添加es5中被支持的尾逗号
all:在有可能的地方都添加
singleQuote 是否使用单引号 false
quoteProps 对象属性中加引号的规则 as-needed as-needed:只在需要的时候给对象属性加引号
consistent:如果对象中有一个属性有引号则都加上引号
preserve:遵循对象属性中引号的用法
jsxSingleQuote JSX中是否使用单引号代替双引号 false 默认情况下不使用单引号代替双引号
jsxBracketSameLine 在多行JSX中最后一行的>是否要另起一行 false <button
id=”btn”
onClick={this.handleClick}
>
Click here
bracketSpacing 在对象字面量的括号(即{和})前后输出空格 true eg:{name:’tom’} => { name: ‘tom’ }
arrowParens 给箭头函数的参数加入括号 always
rangeStart 指定文件需要代码格式化的开始范围 0 默认从文件开头开始格式化
rangeEnd 指定文件需要代码格式化的结束范围 无 默认从文件结尾结束格式化
parser 指定使用哪一种解析器 无 eg: parser: require(“./myParser”)
requirePragma 文件顶部是否需要注释说明文件 false /*
* @prettier
*/
insertPragma 是否在文件顶部插入一个format注释 false 默认不另外插入注释
htmlWhitespaceSensitivity 指定html文件空格的敏感性 css css:遵循css display属性的默认值
strict:空格敏感
ignore:空格不敏感
vueIndentScriptAndStyle vue中script和style标签内的代码是否缩进 false false：不缩进
true：缩进
endOfLine 行尾换行样式 lf lf：仅换行符换行(\n)
crlf：回车符+换行符(\r\n)
cr:仅回车符(\r)
auto：维持文件现有的行尾(通过查看第一行行尾的内容对一个文件中的混合值进行归一化)
