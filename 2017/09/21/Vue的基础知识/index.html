Vue的基础知识
发表于 2017-09-21|study
字数总计:377|阅读时长: 1 分钟
| 阅读量:|评论数:

Vue的三个特点：

  • 易用
  • 灵活（渐进式）

  1. 无论是单页面程序还是多页面程序，我们首先都需要通过声明式渲染来渲染页面上的每一个字段；因为我们的界面需要去展现一些功能，展现一些信息，就需要渲染来实现；
  2. 把一些公共的头部和公共的尾部抽出来去做成一个组件；
  3. 做单页面程序用到路由,需要把vue-resource、vue-router等等插件拉进来去做一个路由的功能；
  4. 如果实现的业务足够复杂，需要用到大量的组件，而且难以管理这些组件的状态，此时就需要引进vuex，用来集中管理组件的状态；
  5. 整个项目做完之后，需要通过构建工具来build系统，提升效率，最后形成一个完整的项目。
  • 高效
  1. vue的min+gzip的运行大小：16kb；
  2. 超快虚拟DOM；
  3. 最省心的优化。

Vue组件的重要选项：

  • data（数据源）
  • methods（方法）
  • watch（监听）

Vue模版指令（html和vue对象的粘合剂）：

  • 数据渲染：v-text 、v-html 、{{ }}

  • 控制模块隐藏：v-if 、v-show

  • 渲染循环列表：v-for

  • 事件绑定：v-bind

  • 表单渲染：v-model: 用于文本框、下拉框、单选复选、textarea等等，具有一个双向绑定的特性。
